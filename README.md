## Laravel Calendar Application

A full-featured calendar application built with Laravel 10, FullCalendar v6, and Tailwind CSS. This application allows users to create, update, and delete events with a modern, responsive interface.

## Features

- Interactive calendar with month, week, and day views
- Create, edit, and delete events
- Color-coded events
- Modal-based event management
- Mobile-responsive design
- Full test coverage

## Requirements

- PHP 8.1 or higher
- Composer
- MySQL or compatible database
- Node.js and npm (for Tailwind CSS)

## Installation

1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```

2. Install PHP dependencies:
   ```bash
   composer install
   ```

3. Install and build frontend assets:
   ```bash
   npm install
   npm run build
   ```

4. Copy the environment file and configure your database:
   ```bash
   cp .env.example .env
   ```

5. Generate application key:
   ```bash
   php artisan key:generate
   ```

6. Run migrations and seed the database:
   ```bash
   php artisan migrate --seed
   ```

## Usage

1. Start the development server:
   ```bash
   php artisan serve
   ```

2. Visit http://localhost:8000/calendar in your browser

## Running Tests

Run the test suite using:
```bash
php artisan test
```

## API Endpoints

- `GET /calendar` - View the calendar
- `POST /events` - Create a new event
- `PUT /events/{event}` - Update an existing event
- `DELETE /events/{event}` - Delete an event
